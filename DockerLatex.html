Docker Latex

View project on GitHub

This discussion will introduce a brief overview about using a built-in Docker Image for creating a Latex file. It also provides some basic Docker command to help the beginners set up a Docker environment

Resources

Go to this web page Docker Images to see a ranked list of Latex Docker images on Docker Hub
Recommended Images: Narf Latex

Set up:

  1. Clone the recommeded images from git hub pages
  2. Set up the docker in your machine. (See the instruction here)
  3. Type "docker pull narf/latex" to build the image
  4. Create a latex file named "input.latex"
  5. Type "docker run -i narf/latex < input.latex > output.pdf" to create an output pdf file
  6. Check to make sure you got everything right
